Branching tutorial 

1. By default our branch is set to master 

2. Chnage the branch from master to other , that simply creates a copy of the master branch , or we can say
it makes a copy of the folder 

3. To create a new branch : git branch  branchname

4. To switch to new branch : git checkout newbranchname

5. To merge teo branches , firsty change the branch to master oon git , then : git merge newbranchname

6. To delete created bracnh  from system we use : git branch -d newbranchname

7. To delete created branch from github we use : git push origin --delete newbranchname
